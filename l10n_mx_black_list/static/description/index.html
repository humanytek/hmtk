Configuración

Black List

Se crean 4 nuevas configuraciones dentro de la contabilidad para controlar el uso de la lista negra. Esto está presente dentro de las Compras, Ventas, Facturas y Contactos; se tiene 3 niveles de control:

  • allow: No ejerce acción alguna
  • deny: Al momento de intentar guardar un registro, se imposibilita esta acción si el contacto está en lista negra.
  • warning: Al momento de modificar un registro, se muetra una alerta si el contacto está en la lista negra.

Funcionalidad

Restringir creación de Contactos

Cuando se crea un nuevo contacto, se verifica que el RFC de este no se encuentre presente en la lista negra; en caso de ser así, se bloquea la posibilidad de crearlo.

Restricción en Facturas, Ventas y Compras

De manera analoga a la creación de contactos, cuando se crea una nueva factura, venta o compra también se comprueba que el Cliente no forme parte de la lista; esto con la finalidad de detectar clientes que anteriormente no estaban en dicha situación.
